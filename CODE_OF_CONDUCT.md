# Code of Conduct

## 우리의 약속

TaskDeck 커뮤니티는 나이, 신체, 장애, 민족, 성별 정체성과 표현, 경험 수준, 국적, 외모, 인종, 종교 또는 성적 정체성과 지향에 관계없이 모두에게 안전하고 존중받는 참여 환경을 제공하기 위해 노력합니다.

## 기대하는 행동

- 서로 다른 관점과 경험을 존중합니다.
- 구체적이고 건설적인 피드백을 제공합니다.
- 실수를 인정하고 공동체에 가장 이로운 방향으로 행동합니다.
- 개인 정보와 보안 신고의 비공개성을 지킵니다.

괴롭힘, 모욕, 위협, 성적 언행, 개인 정보 공개, 지속적인 방해 행위는 허용되지 않습니다.

## 집행

문제가 발생하면 프로젝트 관리 채널을 통해 관리자에게 비공개로 신고해 주세요. 관리자는 신고자의 개인정보를 보호하고 상황에 비례해 경고, 일시 제한 또는 영구 참여 제한 조치를 할 수 있습니다.

이 행동강령은 프로젝트 공간과 프로젝트를 대표하는 공개 공간에 적용됩니다. 본 문서는 Contributor Covenant 2.1의 원칙을 바탕으로 간결하게 작성되었습니다.
