# TaskDeck v0.1.0

Windows에서 앱, 웹사이트, 폴더와 파일을 워크스페이스별로 정리하고 한 번에 작업 환경을 여는 TaskDeck의 첫 MVP 릴리스입니다.

TaskDeck v0.1.0 is the first MVP release of a Windows-first, local-first workspace launcher.

## 주요 기능

- 워크스페이스와 리소스 생성·수정·삭제
- 앱, 웹사이트, 폴더, 파일의 개별 실행 및 순차 열기
- 시작 메뉴, Program Files 및 설치 정보 기반 앱 검색
- `.exe` 직접 선택과 파일·폴더 드래그 앤 드롭
- 분야별 앱·사이트 추천 카탈로그
- JSON 워크스페이스 템플릿 가져오기·내보내기
- 실행 간격, 시스템 트레이, 로그인 시 자동 시작 설정
- 한국어 및 English UI

## 설치 파일

일반 사용자는 NSIS 설치 파일을 권장합니다.

| 파일                           | 용도                         |     크기 | SHA-256                                                            |
| ------------------------------ | ---------------------------- | -------: | ------------------------------------------------------------------ |
| `TaskDeck_0.1.0_x64-setup.exe` | 현재 사용자용 권장 설치 파일 | 2.99 MiB | `91D8B6DE096F220FCF9006D13CE97D00E76C20F852386E9B16A6DCE81186EAA6` |
| `TaskDeck_0.1.0_x64_en-US.msi` | MSI 기반 관리 배포           | 4.23 MiB | `369C5EF523842AA211E18E73BCDDFB73C6DBB5F50C045A58EA73A2A15DF74198` |

## 설치 방법

1. 위 두 파일 중 하나를 내려받습니다.
2. 일반 사용자는 `TaskDeck_0.1.0_x64-setup.exe`를 실행합니다.
3. 설치 후 시작 메뉴에서 TaskDeck을 실행합니다.
4. **설정 → 언어**에서 한국어 또는 English를 선택합니다.

설치 파일은 아직 코드 서명되지 않았으므로 Windows SmartScreen 경고가 표시될 수 있습니다. 파일 이름과 SHA-256을 확인한 뒤 진행하세요.

## 알려진 제한 사항

- Microsoft Store 보호 경로 앱, 일부 특수 런처, 포터블·네트워크 드라이브 앱은 자동 탐지되지 않을 수 있습니다.
- 자동 탐지되지 않은 앱은 `.exe`를 직접 선택하거나 드래그 앤 드롭으로 추가해야 합니다.
- 일부 Windows/Rust 상세 오류는 운영체제가 제공한 원문으로 표시될 수 있습니다.
- 설치 파일은 코드 서명되지 않았고 자동 업데이트는 아직 제공하지 않습니다.

## 검증

이 릴리스 소스에서 다음 검사를 통과했습니다.

```text
npm run format:check
npm run typecheck
npm run lint
cargo test --manifest-path src-tauri/Cargo.toml
cargo check --manifest-path src-tauri/Cargo.toml
npm run tauri build
```

Rust 단위·통합 테스트 15개가 통과했으며 NSIS와 MSI bundle 생성이 완료되었습니다.
